Synchronize wordpress user as your ActiveCampaign email marketing software subscriber.

== Description ==
<p>
Synchronize wordpress user as your <a href="http://www.activecampaign.com/" target="_blank">ActiveCampaign</a> email marketing software subscriber.
Synchronize user by role and also can specify ActiveCampaign list and custom field.
</p>

<p>
Connect your ActiveCampaign Remote application with Authentication tab. 
Set field and list from Synchronize Setting tab. 
Synchronize current user by 'Synchronize Now' button. 
By enabling Auto Synchronize checkbox, keep live update to remote server while user added/edited or deleted. 
</p>


== Installation ==

1. Upload and extract `activecampaign.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You will see new menu named 'ActiveCampaign' under Seetings menu.
